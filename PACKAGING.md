# ResearchMate v2.0 - 产品包装文档

## 📌 项目定位

**ResearchMate** 是一款面向商业作者的智能化素材采集助手，通过系统化采集、智能分类和质量评估，帮助作者快速建立结构化素材库，让深度写作有扎实的内容基础。

### 一句话介绍

> **ResearchMate** —— 让商业作者从繁琐的素材收集中解放出来，把更多精力投入到深度分析和观点提炼上。

---

## 🎯 目标用户画像

### 核心用户群体

| 用户类型 | 典型场景 | 痛点 | 使用频率 |
|---------|---------|------|---------|
| **科技媒体作者** | 撰写行业分析、公司报道、融资新闻 | 找资料耗时、信息分散、难以验证真伪 | 每周 2-3 次 |
| **财经记者** | 深度报道、财报解读、投资分析 | 需要大量数据支撑、时效性要求高 | 每周 1-2 次 |
| **券商分析师** | 行业研究报告、公司点评、投资策略 | 需要跨渠道整合信息、交叉验证 | 每日使用 |
| **企业市场策略人员** | 竞品分析、市场调研、战略规划 | 信息过载、难以筛选高质量内容 | 每月 2-4 次 |
| **自媒体商业内容创作者** | 公众号文章、知乎回答、短视频脚本 | 缺乏权威数据、内容单薄、更新压力大 | 每周 3-5 次 |

### 用户特征

- ✅ 需要频繁撰写商业相关内容（行业分析、公司报道、投资研究等）
- ✅ 对数据准确性要求高，不能容忍事实性错误
- ✅ 时间紧张，需要在短时间内完成高质量内容产出
- ✅ 习惯引用权威来源（财报、研报、主流媒体）
- ✅ 需要系统化管理素材，而非临时搜索

## 💎 核心价值主张

### 三大核心价值

#### 1️⃣ 精准采集 —— 不是简单的关键词搜索，而是理解你的写作意图

**传统方式：**
```
用户搜索："新能源汽车"
→ 得到海量泛泛的结果
→ 需要人工筛选哪些与自己的选题相关
```

**ResearchMate 方式：**
```
用户输入：
  - 选题：人工智能在金融风控中的应用
  - 关键词：同盾科技，百融云创，京东数科
  - 目标字数：5000 字

系统理解：
  - 写作角度：技术落地应用（非行业概览）
  - 聚焦对象：具体公司案例（非泛泛而谈）
  - 素材量级：40000-50000 字（8-10 倍原则）

→ 精准推送相关案例、数据和专家观点
```

**价值：** 减少 70% 无效信息干扰，让每条素材都直接服务于写作目标。

---

#### 2️⃣ 可靠验证 —— 杜绝 AI 幻觉，确保每条数据都有据可查

**传统 AI 工具的问题：**
```
❌ "某新能源车企营收大幅增长"（缺少数值、来源）
❌ "专家认为行业前景广阔"（无具体人名、机构）
❌ "市场规模预计达到万亿"（无时间、无出处）

→ 作者采信后写出的文章会出现事实性错误
→ 严重影响专业性和可信度
```

**ResearchMate 的反幻觉机制：**
```
✅ 四重验证关卡：
   1. 数值检查 - 必须有具体数字
   2. 主体检查 - 必须有公司/产品名
   3. 时间检查 - 必须有清晰时间点
   4. 来源检查 - 必须标注数据来源

✅ 置信度评分 < 0.5 直接剔除
✅ 所有数据都可追溯到原始链接

→ 作者可以放心引用，无需二次核实
```

**价值：** 避免因 AI 胡编乱造导致的文章翻车，保护作者专业声誉。

---

#### 3️⃣ 专业交付 —— 不仅给素材，还给提纲、总结和评估表

**传统方式：**
```
用户得到：一堆零散的链接和文本
→ 需要自己整理归类
→ 需要自己判断哪些可用
→ 需要自己构思写作框架
```

**ResearchMate 交付物：**
```
📁 materials/2026-04-07_人工智能/
├── 人工智能_素材报告.docx    ← Word 格式（可编辑、可打印）
├── 人工智能_素材报告.pdf     ← PDF 格式（方便传阅）
├── 人工智能_素材评估表.csv   ← Excel 表格（勾选标记）
└── materials.md              ← Markdown 格式（网页查看）

包含内容：
  ✅ 文章提纲 - 智能生成的写作框架
  ✅ 核心总结 - 素材概览和使用建议
  ✅ 分类素材卡片 - 每条素材 9 要素完整信息
  ✅ 媒体报道清单 - 原文链接 + 核心观点 + 核心内容
  ✅ 素材评估表 - 评分对比 + 勾选栏 + 补充建议收集
```

**价值：** 从"素材收集"升级到"写作辅助"，让用户拿到就能直接用。
