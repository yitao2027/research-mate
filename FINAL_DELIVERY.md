### 2️⃣ 严格反幻觉机制 ✅

**实现位置**：`src/sources/base_source.py::_verify_data_point()`

**四重验证机制**：

| 验证项 | 检查内容 | 正则表达式 |
|--------|----------|------------|
| **数值检查** | 是否包含具体数据 | `\d+[,.]?\d*` |
| **主体检查** | 是否有公司/产品名 | `[\u4e00-\u9fa5]{2,}` |
| **时间检查** | 是否有时间标识 | `\d{4}[-年]\d{1,2}[-月]\d{1,2}` |
| **来源检查** | 是否标注数据来源 | `据 \| 显示 \| 披露 \| 报告` |

**置信度评分**：
```python
confidence_score = (数值 + 主体 + 时间 + 来源) / 4

if confidence_score < 0.5:
    ❌ 剔除（疑似幻觉数据）
elif 0.5 ≤ confidence_score < 0.7:
    ⚠️ 降级标记（待验证）
else:
    ✅ 通过（高可信度）
```

**新增异常类**：
```python
class VerificationError(SourceError):
    """数据验证失败（反幻觉机制）"""
    pass
```

**实际效果**：
- ✅ 杜绝"某专家表示"等无来源观点
- ✅ 杜绝"据说增长 50%"等模糊数据
- ✅ 所有数据必须有明确来源和具体时间

### 5️⃣ 素材评估表 + 用户反馈 ✅

**评估表（CSV 格式）**：

```csv

...
```

**字段说明**：
- **可信度**（0-1）：来源权威性评分
- **时效性**（0-1）：发布时间新鲜度
- **完整性**（0-1）：数据完整程度
- **交叉验证**（0-1）：多源验证情况
- **综合评分**（0-100）：加权总分
- **等级**（S/A/B/C）：质量分级
- **是否采用**（□）：复选框供用户勾选
- **补充建议**：用户填写需要补充的方向
