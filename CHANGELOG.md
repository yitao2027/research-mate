## [0.1.0] - 2026-04-07

### ✨ 新增功能
- **核心采集器**：支持多源信息采集（新闻、财报、研报、社交媒体）
- **智能分类器**：自动识别素材类型（事实数据、典型案例、专家观点、竞品动态）
- **质量评估器**：多维度评分系统（来源可信度、时效性、完整度、交叉验证）
- **多格式导出**：支持 Markdown、JSON、CSV 三种格式
- **写作灵感生成**：基于素材自动提示潜在选题方向

### 📦 项目结构
- 完整的 Python 包结构（`src/` 模块化设计）
- 配置文件管理（支持用户自定义配置）
- 详细的 README 和使用文档
- GitHub 上传操作指南（新手友好）

### 🎯 使用场景
- 商业文章作者深度素材采集
- 行业研究报告资料收集
- 竞品分析数据采集
- 投资尽职调查信息整理

### 🛠️ 技术栈
- Python 3.9+
- Requests + BeautifulSoup4（网页采集）
- Jieba（中文分词）
- Pandas（数据处理）
- Rich（美化输出）

## 版本说明

### 版本号规则
- **主版本号 (Major)**：不兼容的 API 修改
- **次版本号 (Minor)**：向下兼容的功能性新增
- **修订号 (Patch)**：向下兼容的问题修正

### 变更类型
- **Added**：新增功能
- **Changed**：现有功能的变更
- **Deprecated**：即将移除的功能
- **Removed**：已移除的功能
- **Fixed**：Bug 修复
- **Security**：安全性修复
