### 2️⃣ 反幻觉验证机制

**位置**：`src/sources/base_source.py::_verify_data_point()`

**四重验证**：

```python
def _verify_data_point(self, data_text: str, source_url: str = None) -> Dict:
    # 1. 数值检查 - 是否包含具体数据
    has_numbers = bool(re.search(r'\d+[,.]?\d*', data_text))
    
    # 2. 主体检查 - 是否有公司/产品名
    has_subject = bool(re.search(r'[\u4e00-\u9fa5]{2,}|[A-Za-z]{2,}', data_text))
    
    # 3. 时间检查 - 是否有时间标识
    has_time = bool(re.search(r'\d{4}[-年]\d{1,2}[-月]\d{1,2}|近日 | 近期', data_text))
    
    # 4. 来源检查 - 是否标注数据来源
    has_source = bool(source_url) or bool(re.search(r'据 | 显示 | 披露 | 报告', data_text))
    
    # 置信度评分
    confidence_score = sum([has_numbers, has_subject, has_time, has_source]) / 4
    
    if confidence_score < 0.5:
        return {'valid': False, 'reason': '数据可信度低'}
```

**处理策略**：
- 置信度 < 0.5：直接剔除，不进入素材库
- 置信度 0.5-0.7：降级处理，标记为"待验证"
- 置信度 > 0.7：通过验证，给予高可信度评分

**新增异常类**：
```python
class VerificationError(SourceError):
    """数据验证失败（反幻觉机制）"""
    pass
```
